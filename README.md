## Ejecutar en Dev

1. Clonar repositorio
2. Crear copia de archivo `.env.template`, renombrar a `.env` y agregar la URL del backend
3. Instalar dependencias `npm i`
4. Correr proyecto en modo desarrollo `npm run dev`

## Dependencias

-  `react-icons V5.3.0`
-  `tailwindcss V3.4.15`
